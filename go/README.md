#SwitchRules
This Project used to chrome plugin SwitchSharp VPN Rules

You Can edit rules file from webpage and get SwitchSharp rules info

GO version 1.6.1
complie:
#32bit linux
GOOS=linux GOARCH=386 go build
#64bit linux
GOOS=linux GOARCH=amd64 go build
#32bit windows
GOOS=windows GOARCH=386 go build
#64bit windows
GOOS=windows GOARCH=amd64 go build
#64bit Mac
GOOS=darwin GOARCH=amd64 go build
